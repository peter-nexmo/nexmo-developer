## About
The Nexmo connector for Microsoft Flow allows for multiple actions for you to choose from:

- [Create new application](https://docs.microsoft.com/en-us/connectors/nexmo/#create-new-application)
- [Delete an application](https://docs.microsoft.com/en-us/connectors/nexmo/#delete-an-application)
- [Get application by Id](https://docs.microsoft.com/en-us/connectors/nexmo/#get-application-by-id)
- [List applications](https://docs.microsoft.com/en-us/connectors/nexmo/#list-applications)
- [List inbound numbers](https://docs.microsoft.com/en-us/connectors/nexmo/#list-inbound-numbers)
- [List owned numbers](https://docs.microsoft.com/en-us/connectors/nexmo/#list-owned-numbers)
- [Send a speech file](https://docs.microsoft.com/en-us/connectors/nexmo/#send-a-speech-file)
- [Send an SMS](https://docs.microsoft.com/en-us/connectors/nexmo/#send-an-sms)
- [Update an application](https://docs.microsoft.com/en-us/connectors/nexmo/#update-an-application)
