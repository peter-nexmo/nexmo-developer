Implement the `callNumber:` method to start a call. 

```swift
- (IBAction)callNumber:(id)sender {
    if(!self.ongoingCall) {
        [self startCall];
    } else {
        [self endCall];
    }
}
```

If a call is already in progress, taping the button will end it.
